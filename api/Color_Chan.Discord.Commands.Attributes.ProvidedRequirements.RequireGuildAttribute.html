    Examples

    This example limits all the slash commands in the PongCommands slash command module so they can only be used in guilds. You can also put the RequireGuildAttribute on a method if you only want to have it on a specific command.

    [RequireGuild]
public class PongCommands : SlashCommandModule
{
    [SlashCommand("ping", "Ping Pong!")]
    public Task<IDiscordInteractionResponse> PongAsync()
    {
        // Command code...
    }
}
